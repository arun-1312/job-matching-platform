#  AI Job Matching Platform (Student Project)

A full-stack web application connecting **job seekers** and **employers** with role-based access.  
Developed as part of an internship project and currently **incomplete**.

---

##  Features

-  Job Seeker: Register, login, view jobs, apply  
-  Employer: Register, login, post jobs, view applicants  
-  Role-based authentication  
-  AI-Based Mock interview

---

##  Status

- Core features partially working  
- Authentication implemented  
- Development stopped midway  

---

## ⚠️ Limitations

- No real AI-based matching
